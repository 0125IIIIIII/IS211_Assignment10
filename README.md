Assignment 10 - Music and Pets database

Part 1: A relational database schema for modeling music artists, albums and songs.
Files -
•	music.sql: SQL script to create the music database schema.
•	create_music_schema.py: Python script to create the same schema using SQLite.
•	insert_mock_music_data.py: Inserts sample artists, albums, and songs.
•	query_music_data.py: Queries and displays song data with album and artist info.
•	test_music_schema.py: Unit tests to validate schema and data integrity.
to run the test use: python -m unittest test_music_schema.py 

Part 2: Python scripts for creating, populating, and queryong a pets database using SQLite.
Files -
•	load_pets.py: Creates pets.db and loads sample data.
•	query_pets.py: Interactive CLI to query pet ownership by person ID.
Note: please enter a person ID to view their pets, and enter -1 to exit the query loop.
